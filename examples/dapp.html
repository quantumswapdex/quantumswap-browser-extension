3. Create ERC20 Token

Deploys a bundled ERC20 (see examples/DemoERC20.sol) via qc_sendTransaction with no to. The page ABI-encodes the constructor args and sends data = bytecode ++ abi.encode(name, symbol, supplyWei) together with the abi and bytecode so the wallet can decode and verify (WYSIWYS) before signing. Initial supply is entered in whole tokens and converted to base units (×10¹⁸, 18 decimals) internally. After the deploy confirms, the page reads the receipt back via eth_getTransactionReceipt and fills the new contract address into step 4 below.
